## How to Add Photos

1. **Upload 5 photos** of Nidhi to this directory
2. **Name them**: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`
3. **Supported formats**: `.jpg`, `.jpeg`, `.png`

## File Naming

The photos should be named exactly as:
- `1.jpg` (or `1.png`, `1.jpeg`)
- `2.jpg` (or `2.png`, `2.jpeg`)
- `3.jpg` (or `3.png`, `3.jpeg`)
- `4.jpg` (or `4.png`, `4.jpeg`)
- `5.jpg` (or `5.png`, `5.jpeg`)

## Photo Tips

- **Square photos work best** for the heart-shaped frames
- **Portrait or close-up photos** look great
- **Good lighting** makes the photos stand out
- **File size**: Keep under 500KB each for faster loading

## What Happens

Once you upload the photos:
- They will appear as **floating heart-shaped frames** throughout the quiz
- The hearts will **float up and down** the screen with a gentle animation
- They add a **personal, romantic touch** to the Valentine's quiz
- If a photo fails to load, it will be hidden automatically

## Need Different Names?

If you want to use different filenames, update the `nidhiPhotos` array in `js/config.js`:

```javascript
nidhiPhotos: [
  './assets/images/nidhi/photo1.jpg',
  './assets/images/nidhi/photo2.jpg',
  // ... etc
]
```
